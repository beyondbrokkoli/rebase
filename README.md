VibeEngine is a high-performance 3D rendering framework built on LuaJIT, C, and Vulkan.  
It utilizes a data-oriented architecture to stream 12+ million particles via AVX2 SIMD and Resizable BAR (ReBAR) directly into VRAM.  
The engine features a unique asynchronous render pipeline, custom compute swarm intelligence, and dynamic Vulkan state injectiom.
