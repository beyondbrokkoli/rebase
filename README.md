VibeEngine is a 3D rendering framework  
Built with C, LuaJIT, GLFW and Vulkan  
It utilizes a data-oriented architecture to stream 1+ million particles  
Using AVX2 SIMD and Resizable BAR (ReBAR) streaming directly into VRAM  
The engine features a unique asynchronous render pipeline

Camera & Navigation

    WASD: Move camera forward, backward, left, and right.

    Q / E: Descend and ascend (Y-axis).

    Mouse Look: Move the mouse to rotate the camera.

Engine & Display Controls

    Spacebar: Cycle through available swarm intelligence states (e.g., Tornado, Galaxy, Gyroscope).

    Mouse Buttons: Left-click to push particles; Right-click to pull particles.

    F11: Toggle between native fullscreen and windowed modes.

    Escape: Terminate the engine and initiate a clean teardown.
